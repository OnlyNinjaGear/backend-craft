# Security

This repository contains a backend-safety skill, checker rules, hooks, and
intentionally flawed fixtures. The fixtures are not vulnerable applications to
deploy; they are test corpora.

## Reporting

If you find a security issue in the hook, rules, or install instructions, open
a private report through the repository's security reporting channel when one
is configured. If no private channel exists, contact the repository owner
directly before publishing details.

## Fixture Disclaimer

The projects under `fixtures/` intentionally contain planted production-safety
failures. Do not run them as public services and do not treat them as reference
implementations.
